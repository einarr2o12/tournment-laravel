## 1. Scoring System — BWF 3×15

BWF က 2026 April 25 AGM မှာ အတည်ပြုပြီး 2027 Jan 4 ကစ official ဖြစ်မယ့် format အသစ်။ Sunday Club က ဒီ format နဲ့ ဆော့မယ်။

| Rule | တန်ဖိုး |
|---|---|
| ၁ game / set | **၁၅ မှတ်** |
| Match | **Best of 3** (၂ set အရင်နိုင်သူ ပွဲနိုင်) |
| Scoring | Rally point (rally တိုင်း ၁ မှတ်) |
| Deuce | 14–14 ဖြစ်ရင် **၂ မှတ် ကွာအောင်** ဆက်ဆော့ |
| Hard cap | **၂၁ မှတ်** (20–20 ဆို နောက်ထပ်ရတဲ့ ၂၁ မှတ်နဲ့ ပွဲနိုင်) |
| Mid-game interval | Leader ၈ မှတ်ရောက်ရင် ၆၀ စက္ကန့် နား |
| Game ကြား | ၁၂၀ စက္ကန့် နား |
| 3rd game ends ပြောင်း | ပထမ ၈ မှတ်ရောက်တဲ့ဘက် |

**ဥပမာ — set တစ်ခု ဘယ်လိုနိုင်လဲ:**
- 15–13 → နိုင် ✅ (၁၅ ရောက် + ၂ မှတ်ကွာ)
- 15–14 → မနိုင်သေး (၁ မှတ်ပဲကွာ) → ဆက်ဆော့
- 16–14 → နိုင် ✅ (၂ မှတ်ကွာ)
- 21–20 → နိုင် ✅ (cap ၂၁ ရောက်)

**Group standings:** ၁ ပွဲနိုင် = ၁ မှတ်။ Tie ဖြစ်ရင် → head-to-head → set difference → point difference။
